IMASFilteredRequestBuilder Interface

MASFilteredRequestBuilder defines all of the SCIM operations that are available for SCIM queries.

Syntax

[Android.Runtime.Register("com/ca/mas/identity/common/MASFilteredRequestBuilder", "", "Com.CA.Mas.Identity.Common.IMASFilteredRequestBuilderInvoker")]
public interface IMASFilteredRequestBuilder : Android.Runtime.IJavaObject, IDisposable

Remarks

MASFilteredRequestBuilder defines all of the SCIM operations that are available for SCIM queries. Each Operator type is represented by a specific method that requires only the attribute(s) and the value in order to create a well-formed representation of the Com.CA.Mas.Foundation.IMASRequest serviceable by the SCIM server implementation.
