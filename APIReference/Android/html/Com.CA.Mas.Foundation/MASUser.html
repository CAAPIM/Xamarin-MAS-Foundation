MASUser Class

The MASUser interface is a core interface used to represent a user in the system, as well as an Identity Management definition used in the interaction with SCIM.

Syntax

[Android.Runtime.Register("com/ca/mas/foundation/MASUser", DoNotGenerateAcw=true)]
public abstract class MASUser : Java.Lang.Object, Com.CA.Mas.Identity.User.IMASUserIdentity, Com.CA.Mas.Identity.User.IScimUser, Com.CA.Mas.Messaging.IMASMessenger, IDisposable

Remarks

The MASUser interface is a core interface used to represent a user in the system, as well as an Identity Management definition used in the interaction with SCIM.

The MASUser interface contains the common attribute which is the container for all of the other SCIM values, whether single or multi-valued. SCIM provides a resource type for "User" resources. The core schema for "User" is identified using the following schema URI: "urn:ietf:params:scim:schemas:core:2.0:User". See the for the complete format.

This API supports the Mono for Android infrastructure and is not intended to be used directly from your code.

Syntax

Value

A Type which provides the declaring type.

Remarks

This property is used to control virtual vs. non virtual method dispatch against the underlying JNI object. When this property is equal to the declaring type, then virtual method invocation against the JNI object is performed; otherwise, we assume that the method was overridden by a derived type, and perform non-virtual methdo invocation.
