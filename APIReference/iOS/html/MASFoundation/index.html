MASFoundation.iOS

MASFoundation Namespace

Namespace

Type Description
Constants Contains all string constants and notifications available in the SDK.
MAS The top level MAS object represents the Mobile App Services SDK in it's entirety. It is where the framework lifecycle begins, and ends if necessary. It is the front facing class where many of the configuration settings for the SDK as a whole can be found and utilized.
MASApplication The `MASApplication` class is a local representation of application data.
MASAuthCredentials The `MASAuthCredentials` class is a base class of all other auth credentials types supported in MASFoundation SDK.
MASAuthCredentialsAuthorizationCode The `MASAuthCredentialsAuthorizationCode` class is representation of Authorization code auth credentials.
MASAuthCredentialsBlock The MASAuthCredentialsBlock to provide auth credentials for device registration and/or user authentication.
MASAuthCredentialsJWT The `MASAuthCredentialsJWT` class is representation of JWT auth credentials.
MASAuthCredentialsPassword The `MASAuthCredentialsPassword` class is representation of username/password auth credentials.
MASAuthenticationStatus MASAuthenticationStatus states the authentication status of the application..
MASAuthorizationResponse Alternatively, NSNotification can be subscribed to perform exactly the same behaviour.
MASAuthorizationResponseDelegate MASAuthorizationResponseDelegate protocol is an optional protocol that may be implemented to handle the response of authorization code from social login provider's authorization. These protocols may be implemented in the designated class where social login authentication is being processed.
MASCompletionErrorBlock A standard (bool completed, NSError error) block
MASConfiguration The `MASConfiguration` class is a local representation of configuration data.
MASDevice The `MASDevice` class is a local representation of device data.
MASFile The `MASFile` class is a local representation of file data.
MASFileDirectoryType Enumerated MASFileDirectoryType that indicates which directory to store MASFile into.
MASFoundationErrorCode The enumerated error codes for Foundation level NSErrors.
MASGatewayMonitoringStatus The enumerated MASGatewayMonitoringStatus types.
MASGatewayMonitorStatusBlock The Gateway monitor status block that will receive a MASGatewayMonitoringStatus update when a new status value change is triggered.
MASGrantFlow The enumerated MASGrantFlow.
MASObject The `MASObject` class is a local representation of data persisted to the MAS cloud. This is the main class that is used to interact with objects in your app.
MASRequest MASRequest class is an object created by MASRequestBuilder. It contains all necessary information to invoke an API. The class cannot be constructed or changed directly, only through MASRequestBuilder.
MASRequestBuilder Default configuration value for designated initializer, new MASRequestBuilder("GET/PUT/POST/DELETE"), would be: isPublic: false, sign: false, requestType:MASRequestResponseType.Json, responseType:MASRequestResponseType.Json.
MASRequestResponseType The enumerated MASRequestResponseTypes that can indicate what data format is expected in a request or a response.
MASResponseInfoErrorBlock A standard (NSDictionary responseInfo, NSError error) block. The response object could potentially be any type of object. It is most often used to return NSString JSON responses from HTTP calls for example.
MASResponseObjectErrorBlock A standard (NSHTTPURLResponse response, Object responseObject, NSError error) block. The block will be executed when a request finishes the task unsuccessfully, or successfully. All of three arguments in the block can be null depends on the result of the request, and response object should be casted to appropriate data type when it is received.
MASResponseObjectErrorResult Result of an MASResponseObjectErrorBlock block that will return the NSHTTPURLResponse object, response object which needs to perform type casting based on the object type, and NSError object when error occurs.
MASSecurityConfiguration MASSecurityConfiguration class is an object that determines security measures for communication between the target host.

The class is mainly responsible for SSL pinning mechanism, as well as for including/excluding credentials from primary gateway in the network communication to the target host.

Default configuration value for designated initializer, new MASSecurityConfiguration(new NSUrl("https://your_gateway_hostname:port")), would be: isPublic: false, validateDomainName: true, trustPublicPKI: false.

MASService MASService is the base class for all services which are to be run during the internal service's lifecycle.
MASService_SubclassingHooks MASService subclass Protected Methods.
MASServiceLifecycleStatus The enumerated status for the Lifecycle of the micro services.
MASSharedStorage *Important:* MASSharedStorage will not be available if MASFoundation framework is not initialized; the framework should be initialized prior to write/read/delete any data into MASSharedStorage.
MASState The enumerated MASState that can indicate what state of SDK currently is at.
MASUser The `MASUser` class is a local representation of user data.
MASUserAuthCredentialsBlock The user auth credentials block that will be invoked by SDK to notify developers to provide auth credentials.
MASUserResponseErrorBlock The MASUser specific (MASUser user, NSError error) block
NSData_MAS This category enables encryption methods for the object.
