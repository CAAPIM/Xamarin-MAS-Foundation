MASAuthorizationResponseDelegate Class

MASAuthorizationResponseDelegate protocol is an optional protocol that may be implemented to handle the response of authorization code from social login provider's authorization. These protocols may be implemented in the designated class where social login authentication is being processed.

Syntax

[Foundation.Model]
[Foundation.Register("MASAuthorizationResponseDelegate", false)]
public abstract class MASAuthorizationResponseDelegate : Foundation.NSObject
