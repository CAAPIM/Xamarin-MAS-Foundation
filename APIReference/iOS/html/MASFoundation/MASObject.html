MASObject Class

The `MASObject` class is a local representation of data persisted to the MAS cloud. This is the main class that is used to interact with objects in your app.

Syntax

[Foundation.Register("MASObject", true)]
public class MASObject : Foundation.NSObject, Foundation.INSCoding, Foundation.INSCopying, IDisposable

SetObjectForKey Method

Sets the object associated with a given key.

Syntax

[Foundation.Export("setObject:forKey:")]
public virtual void SetObjectForKey (Foundation.NSObject object, Foundation.NSCopying key)

Parameters

object
The object for `key`. A strong reference to the object is maintaned by MASObject. Raises an `NSInvalidArgumentException` if `object` is `null`. If you need to represent a `null` value - use `NSNull`.
key
The key for `object`. Raises an `NSInvalidArgumentException` if `key` is `null`.
