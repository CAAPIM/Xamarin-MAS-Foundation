◆ LockSessionWithCompletion()

void MASFoundation.MASUser.LockSessionWithCompletion ( [NullAllowed] MASCompletionErrorBlock  completion)

Lock currently authenticated user session with device's local authentication (Device's Passcode or Biometric Authentication).

This will lock current user session and block most of operations except for MASUser.CurrentUser.LogoutWithCompletion(); and MASDevice.CurrentDevice().DeregisterWithCompletion(). Alternatively, locked user session can be removed by MASUser.CurrentUser.RemoveSessionLock() which will remove credentials from protected stroage of the device with local authentication.

After locking user session, MASUser.CurrentUser will still exists; however, MASUser.CurrentUser.IsAuthenticated property will reflect the lock status and will return false until it's unlocked. Session lock status can also be checked through MASUser.CurrentUser.isSessionLocked property.

Parameters
completionThe MASCompletionErrorBlock block that receives the results.
